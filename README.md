## English

**Hakoniwa Kreo** is a Windows app for building 2D games with an intuitive,
no-code **event-sheet** workflow. Lay out "if this, then that" rules to create
game logic — tile maps, animations, physics, and cameras all included. Games you
make can be **sold commercially even on the Free plan**, and exported apps run
**permanently**, regardless of license status.
